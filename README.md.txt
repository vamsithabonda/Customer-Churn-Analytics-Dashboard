# Customer Churn Analytics Dashboard

## Project Overview

This Power BI dashboard analyzes customer churn patterns and provides actionable insights to improve customer retention.

## Tools Used

- Power BI
- DAX
- Power Query
- Excel / CSV

## Dashboard Pages

### Executive Overview

- Total Customers
- Churned Customers
- Churn Rate
- Revenue
- Revenue Lost

### Customer Segmentation Analysis

- Gender Analysis
- Senior Citizen Analysis
- Partner Analysis
- Dependents Analysis
- Tenure Analysis

### Customer Retention Insights

- Contract Type
- Online Security
- Tech Support
- Payment Method

## Key Insights

- Month-to-Month customers have highest churn.
- Customers without Tech Support churn more.
- Fiber Optic users show higher churn.
- Lack of Online Security increases churn.
